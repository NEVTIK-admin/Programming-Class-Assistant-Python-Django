Angka terbilang


ubah angka dari input menjadi angka yang terbilang

contoh:

kevin@Samsung:~$ python angka.py
12345
seratus dua puluh ribu tiga ratus empat puluh lima
kevin@Samsung:~$


aturan:
input jangan ada message apapun (gunakan saja input() tanpa argumen)

outputnya harus berbahasa indonesia, huruf kecil semua, tidak ada tambahan apapun
